# Building AI Project

## Summary
Ushbu loyiha sun'iy intellekt yordamida ob-havo suratlarini (quyoshli, yomg'irli, qorli) avtomatik aniqlashga mo'ljallangan.

## Background
Bu loyiha qishloq xo'jaligi dronlari va avtopilot mashinalar uchun tashqi muhitni yaxshiroq tushunishga yordam beradi.

## How it works
Loyiha Convolutional Neural Networks (CNN) modelidan foydalanadi. U rasmlarni piksellar bo'yicha tahlil qilib, ehtimollik natijasini chiqaradi.
